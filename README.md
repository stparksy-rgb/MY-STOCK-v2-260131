# AI 주식 트레이딩 시스템 v2

## 🎉 새로운 기능

### ✨ v2 업데이트 (2025-01-31)
- ✅ 구글 시트 테마별 종목 관리
- ✅ 멀티 테마 선택 기능
- ✅ 자동 종목 로드
- ✅ 테마별 종목 수 표시

## 📊 기능

### 차트 분석
- 한국 주식 실시간 분석
- 스토캐스틱 + RSI 지표
- AI 매수/매도 신호
- 이동평균선 (5/20/60일)

### 구글 시트 연동
- 테마별 종목 관리
- 여러 테마 동시 선택
- 자동 업데이트 (10분마다)
- 종목 무제한 추가 가능

### 백테스팅
- 과거 데이터 기반 전략 검증
- 승률 및 손익비 계산
- 거래 내역 분석

## 🚀 사용 방법

1. **직접 입력 모드**
   - 종목 코드 직접 입력 (예: 005930, 000660)

2. **구글 시트 모드**
   - 구글 시트 URL 입력
   - 테마 선택 (여러 개 가능)
   - 자동으로 종목 로드

## 📋 구글 시트 형식

```
테마구분 | 종목코드
---------|----------
시가총액  | 005930, 000660
전기차    | 373220, 006400
```
